# 🧠 JS-INNOV.IA — Formulaire d'enregistrement automatisé

Mini application professionnelle développée avec **Next.js**, **Tailwind CSS**, et intégrée à **Airtable** et **Make** pour automatiser la gestion des leads (formulaires auto, incendie, épargne...).

## 🔧 Fonctionnalités
- Formulaire intelligent avec enregistrement automatique
- Intégration avec Airtable
- Compatible avec l’upload de fichiers (PDF, images…)
- Adapté aux assurances (auto, incendie, etc.)
- Déployé sur **Vercel**

## 🚀 Lancer le projet en local
```bash
npm install
npm run dev
